У меня есть два файлa: "writer.c" и "reader.c". Идея заключается в следующем: "writer.c" получает входной текст в качестве аргумента, а затем записывает его в файл буфера путем разделения на группы по 5 символов. "reader.c" программа считывает данные из этого файла буфера.

Входной текст должен быть введен как одно слово через аргумент командной строки для программы "writer".
